# Nexaflow WordPress Theme

**Nexaflow** — это кастомная тема для WordPress, полностью натянутая с готового сайта.  

## Демо

Вы можете посмотреть живую демку по ссылке: [Nexaflow Demo](https://mrjohnny-code.github.io/nexaflow/)  


## Структура темы

- `css/style.css` — основной CSS  
- `functions.php` — регистрация CPT и подключение стилей/шрифтов  
- `template-parts/` — отдельные блоки сайта (hero, about, benefits, FAQ и др.)  
- `footer.php` и `header.php` — шапка и подвал сайта
